# Bayesian Network Pathfinding System

## Description
An intelligent pathfinding system that uses Bayesian Networks to navigate a grid-based environment with obstacles.

## Technologies
Python, Bayesian Networks, Tkinter

## Features
- Probabilistic decision making
- Shortest path calculation
- Real-time visualization

## Results
Successfully finds optimal paths under uncertainty using probabilistic inference.

## How to Run
1. Install Python
2. Run main.py
